##Examforge is a data-driven and domain specific webapp that in its current stage covers Engineering physics as per the syllabus of RGPV bhopal Semester 1/2. Examforge uses pre-defined datasets made from reliable sources to get an idea of the query asked by the user then uses GeminiAI to fetch real, accurate and reliable explanations as per the RGPV Format.
#Purpose- Examforge helps students be exam ready it has to the point explanations and has a feature that allows the user to generate exam ready answers making learning go smoothly.
